# TES-platform Batch Scheduler Admin Dashboard

This project is a React-based admin dashboard with routing and TailwindCSS for styling. It includes components for a header, sidebar, and main content area, as well as routing for different pages using React Router.

## Getting Started

### Prerequisites

Ensure you have the following installed:

- Node.js (version 20 or later)
- Yarn package manager

### Installation

1. Clone the repository:

   ```bash
   git clone https://git3.fsoft.com.vn/GROUP/GLCSKCC2024/tes-platform/fe-batch-scheduler.git
   cd fe-batch-scheduler
2. Install dependencies:
    ```bash
    yarn install
3. Building the Project:
    ```bash
    yarn build
4. Running the Development Server:
    ```bash
    yarn dev    
5. Running Test
    ```bash
    yarn test
6. Running EsLint
    ```bash
    yarn lint 
7. Running Prettier to Format code
    ```bash
    yarn format
